# Calculator (Version Alpha 1.1)

A modern graphical desktop calculator built with Python using the tkinter and customtkinter libraries. 

## Installation and Usage
To run the pre-compiled application on Windows, navigate to the project directory and double-click the executable file:
calculator_Alpha_v1.1.exe

## Technical Details and AI Collaboration
The initial graphical user interface (GUI) layout, custom widgets, and the visual design system using the customtkinter library were generated with the assistance of Artificial Intelligence. 

The complete codebase has been thoroughly refactored, optimized, and verified by the author. All internal mathematical functions, event handling, and input validations have been customized to ensure correct logical flow. The source code (calculator_v1.1.py) is fully open and available for modifications, redistribution, or code review.
